# Landing Page

Homepage at `/`. Hero section, featured products, latest blog posts, random curiosity.

## Sections

1. **Hero** — shop name, tagline, CTA to products
2. **Featured products** — up to 6 products (inStock), from ProductRepository
3. **Latest posts** — up to 3 published posts, from PostRepository
4. **Curiosity** — one random curiosity from CuriosityRepository

Uses ShopSettings for shop name, logo, tagline.
